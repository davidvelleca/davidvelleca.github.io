Fun with Set Actions

Posted by David Velleca on April 29, 2019

Tabyrinth - An Interactive Tableau Game

So, my love of set actions continues. In November, Ken Flerlage posted a connect the dots on Tableau Public. Then in December, Kevin Flerlage posted a holiday themed connect the dots. One of my favorite things in Tableau is the interactivity that the tool can drive. So, inspired by those two vizzes, I created an interactive crossword puzzle. This set the stage for more creativity...

Earlier this year, I had the idea to create a maze in Tableau where the user works their way through the maze with their mouse and hitting the wall drives a message saying something like 'Try Again.' At the time, my understanding of set actions wasn't what it is today, so I put the idea on hold. Lately, I've been using set actions for all types of things, and becoming incredibly comfortable with them. So, I dusted off the idea and took a stab at it.
